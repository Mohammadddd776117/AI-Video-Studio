# Tool Schema Package

## Purpose

This directory contains the tool schema foundation for AI Video Studio.

## Ownership

Owned by the AI execution and tool contract layer. It should remain the authoritative source for tool capabilities and action execution contracts.

## Responsibilities

- tool capability definitions
- tool invocation structure
- bounded execution contract references
- provider-neutral tool registration rules

## Future Implementation Scope

This package will evolve into the reusable tool contract foundation used by the AI orchestrator and agent runtime.
