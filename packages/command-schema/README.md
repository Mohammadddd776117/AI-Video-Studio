# Command Schema Package

## Purpose

This directory contains the command schema foundation for AI Video Studio.

## Ownership

Owned by the AI command layer and shared contract model. It should remain the authoritative source for the structured command and request representation used by the AI system.

## Responsibilities

- command schema definitions
- command validation structures
- versioned request and response shape references
- AI execution contract stability

## Future Implementation Scope

This package will evolve into the canonical command contract used across orchestration, agent runtime, and downstream tool execution components.
