# Data Models

## Overview
This document defines the core data models used across AI Video Studio.

## User Model
- User identifier
- Account profile and preferences
- Subscription tier and entitlements
- Authentication and security settings
- Usage history and billing context

## Workspace Model
- Workspace identifier
- Owners, members, and roles
- Shared settings and collaboration policies
- Connected integrations and API keys

## Project Model
- Project identifier
- Workspace ownership
- Timeline references
- Status and version history
- Export and publishing metadata

## Timeline Model
- Timeline identifier
- Tracks and clips
- Sequence ordering and timing
- Effects, captions, and annotations
- Versioned edit state

## Asset Model
- Asset identifier
- Media type and storage location
- Metadata and analysis results
- Derived proxies and thumbnails
- Ownership and permissions

## AI History Model
- Request identifier
- Prompt and structured command reference
- Model version and tool version
- Execution state and outcome
- Approval decisions and audit trail

## Collaboration Model
- Comment threads and review states
- Permission grants and shared access
- Notification and assignment metadata
- Version comparison and change history
